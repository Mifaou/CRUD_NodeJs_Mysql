<h2>Desc: </h2>
This app allows you  to Add / Update / Delete user and all users is stored in a mysql database.
Using :
Mysql
Nodejs
Express
Ejs
1- Create a Mysql DB called form
2-Import DB ( test.sql)
2- Open config.js and insert your personal username and password for Mysql
3- Open terminal in your current folder and install packages (Node Modules) ( i shared with u in case )
4- Run your server using node app
5- Open your browser and put http://localhost:3000( you can change port if you want )
6- Finally your app is ready !!
